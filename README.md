# Exploratory Data Analysis and Data Visualization

## Problem Statement

The objective of this task is to perform Exploratory Data Analysis (EDA) on a dataset to identify patterns, trends, and meaningful insights through statistical analysis and data visualization.


## Dataset Details

* The dataset contains both categorical and numerical features.
* It includes information such as content type, genre, country, rating, release year, duration, and view statistics.
* The dataset was analyzed using Python and Pandas.


## Approach

The following steps were performed:

1. Imported and inspected the dataset.
2. Checked dataset shape, data types, and missing values.
3. Generated summary statistics.
4. Created visualizations, including:

   * Movies vs TV Shows Count Plot
   * Genre Distribution
   * Country-wise Content Distribution
   * Ratings Distribution
   * Duration Histogram
   * Correlation Heatmap
5. Extracted key insights such as:

   * Most common genre
   * Most common country
   * Most common rating
   * Average duration
   * Average views

## Tools & Libraries**

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab

## Results

* Successfully explored and analyzed the dataset using EDA techniques.
* Identified content distribution across different categories.
* Visualized trends and feature relationships using multiple charts.
* Generated summary insights to better understand the dataset.
* Strengthened practical skills in data analysis and visualization using Python.
